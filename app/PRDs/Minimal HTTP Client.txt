Objective
Build a minimal but complete HTTP Client that enable users to:

Create and organize API requests inside collections

Author and edit HTTP requests

Execute requests and view responses

Work on multiple requests simultaneously using tabs

This release focuses on delivering a complete, minimal request authoring and execution experience, without advanced workflow or productivity tooling.

Success Criteria
User can:

Create collection

Create request

Edit request

Save request

Send request

View response

Open multiple requests in tabs
