Dynamic Status Code Test

Setup Instructions:

  1. Load the built extension from browser-extension/mv3/dist/ in chrome://extensions
  2. Create a Modify Response rule in Requestly with:
    • URL Contains: httpbin.org
    • Response Type: Dynamic (JavaScript)
    • Code:
    function modifyResponse(args) {
  const {method, url, requestData, responseJSON} = args;

  // Dynamically set status code based on URL path
  if (url.includes("/status/")) {
    args.statusCode = 403;
    return {"error": "Forbidden", "dynamic": true};
  }

  args.statusCode = 201;
  return {"message": "Created dynamically", "originalUrl": url};
}
  3. Click the test buttons below
